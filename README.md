# Real-time-Face-Mask-Detection



## Used Technology
* Tensorflow
* Keras
* Open CV2 


## Getting Started

### Dependencies

* [requirement.txt](https://github.com/iqbalsublime/Real-time-Face-Mask-Detection/blob/main/requirements.txt)

#### Train 

```

```

```
$python my_model_local_video.py
```
